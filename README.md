## GIF Player

[GIFPlayer](https://github.com/PepperoniJoe/GIFPlayer) - A Swift app that plays GIF files as animated images.

UIImage.animatedImage, color assets\
Apple Keynote: magic move, rotate, shimmer, bounce, jiggle
